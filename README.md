Weather App 🌤️
A simple and elegant weather application that allows users to search for the current weather conditions of any city. Built with HTML, CSS, and JavaScript, this app fetches real-time weather data using the OpenWeatherMap API.



Features

Search by City: Enter a city name to get the current weather conditions.
Weather Details:
Temperature (°C)
Feels Like Temperature (°C)
Weather Description (e.g., Clear Sky, Rain)
Humidity (%)
Pressure (hPa)
Wind Speed (m/s)
Weather Icons: Visual representation of the current weather.
Responsive Design: Works seamlessly on both desktop and mobile devices.
Modern UI: Clean and professional design with animations and a glassmorphism effect.


Technologies Used

HTML: For the structure of the website.
CSS: For styling and animations.
JavaScript: For dynamic functionality and API integration.
OpenWeatherMap API: For fetching real-time weather data.
How to Use

Search for a City:
Enter the name of a city in the input field.
Click the "Search" button or press Enter.
View Weather Details:
The app will display the current weather conditions, including temperature, humidity, wind speed, and more. -->
